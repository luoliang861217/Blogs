Gentoo xfce4 下无法设置 jpg 文件为壁纸

  • May 31, 2014

Ristretto 也无法打开 jpg 文件

可以确定某个包没有添加 jpeg 这个 USE 导致的

使用 equery h jpeg 查看哪些包拥有 jpeg 这个 USE

然后挨个 eix 看看每个包都什么用吧，我已经找出来了，是 gdk-pixbuf

现在 echo 'x11-libs/gdk-pixbuf jpeg' >> /etc/portage/package.use

然后emerge -qva gdk-pixbuf 重新安装

最后注销一下 slim 重新登入

备注：

  • feh 无法浏览 jpg 文件并设置为桌面，是 imlib2
  • awesome 自带设置壁纸功能，如果不能设置 jpg 文件为壁纸，是 imagemagick
  • scrot 无法将截图保存为 jpg 文件时，也是 imlib2

Typecho 添加 sitemap.xml

  • May 5, 2014

首先去 Hanny 的博客下载 sitemap 插件，解压到 /usr/plugins/Sitemap

修改 Plugin.php 中 activate 方法为

Helper::addRoute('sitemap', '/sitemap.xml', 'Sitemap_Action', 'action');

修改 Action.php 中 action 方法，修改 header 和 foreach 之间内容如下

echo "<?xml version=\"1.0\" encoding=\"utf-8\"?>\n";
echo "<?xml-stylesheet type='text/xsl' href='" . $options->pluginUrl . "/Sitemap/sitemap.xsl'?>\n";
echo "<urlset xmlns:xsi=\"http://www.w3.org/2001/XMLSchema-instance\"\n
              xsi:schemaLocation=\"http://www.sitemaps.org/schemas/sitem
ap/0.9 http://www.sitemaps.org/schemas/sitemap/0.9/sitemap.xsd\"\n
              xmlns=\"http://www.sitemaps.org/schemas/sitemap/0.9\">";

然后 戳 gists 下载 sitemap.xsl 放在插件目录下，这样就 Ok 了

Arbitrary 添加归档页面

  • May 5, 2014

有人问到归档页面，这里说一下吧，还是利用 Typecho 的自定义模板功能

在主题目录下新建一个 PHP 文件，名字无所谓，我取的是 tp-archive.php 表示这是个 archive template，文件中写入以下内容

<?php
/**
 * Archive Page
 *
 * @package custom
 */

    $this->need('header.php');
?>

<section class="article-list col-tb-8 col-mb-12">
    <section id="breadcrumb">
        <a href="<?php $this->options->siteUrl(); ?>">首页</a>
        <span>&gt;</span>
        <?php if ($this->is('post')): ?>
            <?php $this->category(); ?>
            <span>&gt;</span>
            <?php $this->title() ?>
        <?php else: ?>      
            <?php $this->archiveTitle('','',''); ?>
        <?php endif; ?>
    </section>

    <?php
    $this->widget('Widget_Contents_Post_Recent', 'pageSize=10000')->to($archives);   
    $output = '<article><ul id="archives">';

    while ($archives->next()) {
        $output .= '<li><span>' . date('Y-m-d', $archives->created) . '</span><a href="'.$archives->permalink .'">' . $archives->title . '</a></li>';
    }

    $output .= '</ul></article>';
    echo $output;
?>
    <?php $this->need('comments.php'); ?>
</section>

<?php
    $this->need('side.php');
    $this->need('footer.php');
?>

Ruby基础语法(2): Object More

  • May 2, 2014

上文补充

在 Ruby基础语法(1) 中有一些没有提到，这里补充一下。

{} 与 do end

前面说过，{} 的优先级比 do end 高，没有详细介绍，这里给个示例证明一下。

# 使用 *arg 吸收参数以消除参数不匹配的警告
def first *arg
    yield "first" if block_given?
end
def second *arg
    yield "second" if block_given?
end

first second { |str| puts str }
# second
first second do |str| puts str end
# first

可以看出，{} 是跟最右边的方法调用关联，而 do end 则是跟最左边的方法调用关联。

SASS 用法简单介绍

  • April 28, 2014

前言

最近用了一点 Sass，看了一点官方资料，把一些简单的用法整理如下。

Sass 是什么

Sass 是一个基于 Ruby 语言的内部 DSL，可以被编译成 css 文件，使用 Scss 可以方便的创建，修改 css 文件。

Sass 有两种语法格式，一种是像 python 那样使用缩进代表层次的 sass，个人不喜欢这种方式，另一种就是本文介绍的 scss。

Scss 使用

Sass 是基于 Ruby 的，所以如果你想在自己的电脑中使用 Scss，你需要先安装 Ruby。

安装 Ruby

Linux 下可以使用 pacman emerge apt-get yum 等包管理器安装，Windows 下可以用二进制安装包，你还可以自己手动编译。

网上有许多文章说千万不要在 Windows 上使用 Ruby，理由 1 2 3，我没用过，不发表看法。

安装 Sass

% gem install sass
